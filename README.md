# yanghan.dev

Personal website source for <https://yanghan.dev>.

## Structure

- `_pages/` contains the top-level site pages.
- `_posts/` contains blog posts.
- `_bibliography/papers.bib` powers the publications page.
- `_data/resume.json` powers the CV page.
- `assets/` contains the static files used by the live site.

## Deploy

GitHub Actions builds and deploys the site from `.github/workflows/deploy.yml`.
